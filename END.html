Nossa jornada

Inicialmente a equipe pensou em usar a API de 'STAR WARS' porém ela não estava funcionando foi quando, depois de tanto pensar a gente teve a brilhante ideia de usar a API de ‘STRANGER THINGS’, ela foi mais acessível e pratica para fazermos o uso dela a gente acessou inicialmente o GIT HUB e começou a nossa aventura. Nossa API faz o uso de número para acessar as informações dos personagens. É uma pesquisa rápida com um número aleatório que retorna o nome de um personagem e seu lema.

Dados técnicos

A manipulação dos dados foi feito através do JavaScript
  • Busca de informações do personagem: A função fetchCharacterInfo é definida para buscar informações de um personagem da API strangerthings-quotes. Ela usa a função fetch para fazer uma solicitação GET para a API e passa o número do personagem como parte da URL. A resposta é então convertida em JSON e retornada.
  • Criação de usuários: A função create é definida para lidar com o evento de envio do formulário. Quando o formulário é enviado, ela impede o comportamento padrão do formulário (que é recarregar a página), obtém o valor do campo de entrada “name”, e então chama a função fetchCharacterInfo para obter as informações do personagem
  • Exibição de informações do personagem: Se a chamada para fetchCharacterInfo for bem-sucedida, o código cria um novo elemento div, adiciona a classe “userItem” a ele, cria um novo elemento p com as informações do personagem e adiciona esse elemento p ao elemento div. Em seguida, ele adiciona o elemento div ao elemento userList.
